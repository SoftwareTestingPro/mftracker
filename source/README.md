# mftracker

Mutual Funds Tracker (mftracker) - A secure, anonymous way to track your mutual fund investments using Google Drive for private storage.

## Features

- **Secure & Private**: Your data is stored in your own Google Drive account.
- **Anonymous**: No personal data is collected by the app.
- **Easy Tracking**: Track your mutual fund portfolio performance with ease.
- **Modern UI**: Built with Flutter for a smooth cross-platform experience.

## Getting Started

To run this project locally:

1. Clone the repository.
2. Ensure you have Flutter installed (`flutter doctor`).
3. Run `flutter pub get`.
4. Run `flutter run`.

## Build & Deploy

### Web
Run `build_web.bat` to build and deploy to the root directory for GitHub Pages.

### Android
Run `build_android.bat` to generate a release APK.
